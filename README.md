# Wine Quality EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Wine Quality dataset to understand which features affect wine quality.

EDA helps us:
- Understand data
- Find patterns
- Detect relationships between variables
- Prepare data for Machine Learning

---

## Dataset Features
The dataset contains chemical properties of wine such as:

- Fixed acidity
- Volatile acidity
- Citric acid
- Residual sugar
- Chlorides
- Free sulfur dioxide
- Total sulfur dioxide
- Density
- pH
- Sulphates
- Alcohol
- Quality (target variable)

---

---

## Analysis Performed
- Data loading
- Data cleaning
- Statistical summary
- Correlation heatmap
- Pairplot visualization
- Feature analysis
- Insights generation

---

## Project Goal
To identify important chemical properties that influence wine quality and prepare dataset for machine learning models.
